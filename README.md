# kakaoip-front

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
### Lints and fixes files
```
yarn lint
```

### Compiles for development
```
yarn build:dev
```
### Compiles for staging
```
yarn build:cbt
```
### Compiles for production
```
yarn build:prod
```

### Start Server
```
yarn start
```
